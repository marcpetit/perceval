# Projet PERCEVAL

**PERCEVAL : Plateforme de jeu d'Echecs Réduite pour la Compréhension, l'Expérimentation et la Vérification en Architecture Logicielle**

Ce projet de plateforme de jeu d'échecs est avant tout un terrain d'expérimentation pour servir de support à des expérimentations en architecture logicielles menées dans le cadre de vidéos à but pédagogique. Il n'a donc pas vocation à livrer une plateforme utilisable. Qui plus est, dans ses premières versions il mettra de plus en oeuvre des choix très discutables, qui ont justement pour objectif d'être discutés et évalués par le suite. Il est donc déconseillé de vous en inspirer pour réaliser un projet réel.
